See it in action

Hello GPU
let a = gpu_fill(1.0, 1000000)
let b = gpu_fill(2.0, 1000000)
let c = gpu_add(a, b)
let total = gpu_sum(c)
print("1M elements: {total}")
Functions & Lambdas
let nums = [1, 2, 3, 4, 5]
let evens = filter(nums, fn(x) x % 2 == 0 end)
let squared = map_each(evens, fn(x) x * x end)
let total = reduce(squared, 0, fn(a, x) a + x end)
print("Sum of squares: {total}")
Data Analysis
let data = read_csv("sales.csv")
let revenue = csv_column(data, "revenue")
let avg = mean(revenue)
let std = stddev(revenue)
print("avg={avg}, std={std}")
Stream Pipelines
stream photo = tap("input.jpg")
stream warm = photo
    |> brightness(20)
    |> contrast(1.2)
    |> saturate(1.1)
emit(warm, "output.png")

Why OctoFlow

⚡

GPU-First

The GPU is the primary execution target. Data born on the GPU stays on the GPU. CPU handles I/O.

📦

Zero Dependencies

One 2.2 MB binary. No SDK, no runtime, no package manager. Download, unzip, run.

🌐

Any GPU Vendor

NVIDIA, AMD, Intel. Runs on any GPU with a Vulkan driver. No vendor lock-in.

🔒

Sandboxed by Default

No file access, no network, no subprocesses unless you explicitly allow them with flags.

🧠

AI-Friendly

Language guide designed as LLM context. Feed it to your AI assistant and it can write OctoFlow code.

🏗️

Self-Hosted

48% of the compiler is written in OctoFlow itself. The language compiles itself.

Get started

🖥️ Windows x64 🐧 Linux x64
